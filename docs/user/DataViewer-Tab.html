DATA VIEWER:

This is the main view to load, save, edit and inspect data. The data viewer is based on plot panels, which can be added to the view and then configured.


Figure 1: Example data viewer layout

View Panels:

A panel is the the frame around the enclosed axes. Panel options can be changed by right-click on the panel frame (outside the plotting axes). The panel options context menu will only show, if all plot view options (Zoom, Pan, ...) are disabled (Arrow button). The currently active panel is indicated by a cyan frame. Change the active panel by left-click onto another panel.A single panel can be deleted from the panel context menu.

The default panel is a time-plot, which creates plotting axes to visualise one or more data channels against time. Other panel types available are map-plot, correlation-plot and spectrum-plot. Some plot types require a link to an existing time-plot, where the data of the time-plot is used as the data source for the secondary plot. Changes to the time-plot content are then automatically reflected in the secondary plot. The linking procedure works by selecting a new plot option from a panel context menu, and then left-clicking into the desired, existing time-plot panel. Once linked, the panel frame colours change to indicate the link, as show in Figure (1) on the right.

Add plot/panel:

Creates a new plot panel at the bottom of the current column ( the column with the active panel ). The new panel defaults to time-plot.

Add column:

Creates a new panel column on the right with a new plot panel inside. The new plot panel becomes the new active panel.

Clear:

Deletes all plot panels except for the top left panel.
