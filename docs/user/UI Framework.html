UI Framework
=======================

Basic UI elements common to all tabs

Menu:


UI Controls Common Ribbon:


Open existing FDS file:

Function: Opens an existing FDS file from disk. The data format is upgraded to the default FDS format, if required.


Save FDS:

Function: Save FDS file to user selected location.

Select:

Function: Enables context menus for each line in the plot (disables all other plot manipulation buttons).


Zoom Toggle:

Function: Enable Matlab zoom function. The right click context menu has been customised with the following options:

  • Undo Zoom: Restores plot limits to view before last zoom operation
  • Zoom to fit: Zooms to global data range, identical to Zoom to fit button
  • Zoom In/Out: Select zoom direction
  • Unconstrained: Zoom X and Y axes
  • Horizontal: Zoom X axis only
  • Vertical: Zoom Y axis only


Zoom to fit:

Function: Zoom all axes to fit the plotted data global limits.


Pan Toggle:

Function: Enables native Matlab pan function. Plot limits are updated during pan operation.


Data Cursor Toggle:

Function: Enables native Matlab data cursor function


Hold Toggle:

Function: Add lines to existing plot when depressed (‘hold on’) – applies to selected axis


Clear:

Function: Clear and reset all axes
